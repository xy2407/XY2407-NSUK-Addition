# XY2407 NSUK Addition

**NeoForge 1.21.1** — An expansion mod for New:Sim-U-Kraft, adding breeding, mining, ore veins, city upgrades, auto-restock, winery, and more.

| Property | Value |
|----------|-------|
| **Mod ID** | `xy2407_nsuk_addition` |
| **Platform** | NeoForge 1.21.1 |
| **Author** | xingying2407 |
| **License** | GPL-3.0 |

***

## Features

### Breeding

- **Breeding Control Box** — Multi-block controller for managing animal and fish breeding
- **Fish Breeding Support** — Compatibility with LetFishLove Reborn, Tide, and Aquaculture
- **Roe Block** — Dynamic roe block supporting multiple fish species
- **Breeding Definition Loader** — Custom breeding recipes via JSON config
- **Building Deployment** — Auto-deploy fish pond / farm building packs

### Mining

- **Mining Control Box** — Multi-block mining controller
- **Mining Menu** — Full GUI menu with LDLib2 UI
- **Player Manual Mining** — Manual mining support

### Ore Vein

- **Ore Vein Generation** — Natural ore vein distribution during world generation
- **Ore Vein Discovery** — Players can discover and register ore veins
- **Ore Vein Highlight** — Xaero's World Map compatible vein highlighting
- **Rich Iron Ore Rendering** — Rich iron ore overlay rendering
- **Drop Handling** — Vein-specific drop processing

### City Upgrade

- **City Level System** — Multi-level city upgrade support
- **Upgrade Requirements** — Population, resources, funds condition checks
- **Upgrade Panel** — Replaces vanilla Sim-U-Kraft upgrade UI
- **Core Relocation** — Support moving city core position

### Auto Restock

- **Industrial Control Box Auto Restock** — Auto-refill raw materials and fuel for industrial control boxes
- **Commercial Control Box Auto Restock** — Auto-restock goods for commercial control boxes
- **SQLite Persistence** — Persistent storage of restock configurations

### Farmland & Crops

- **Tall Crop Support** — Support for sugarcane, bamboo, etc.
- **Vines & Bushes** — Grapevines, grape bushes, etc.
- **Rice** — Rice farming support
- **Right-Click Harvest** — One-click harvest compatible crops
- **Multi-Mod Crop Support** — Compatible with Farmers Delight, Farm & Charm, Cultural Delights, Kaleidoscope Cookery, etc.

### Sidebar HUD

- **Info Sidebar** — Configurable on-screen sidebar information display
- **City Data Panel** — Display city population, funds, and stats
- **Building Statistics** — Statistics for various building types
- **Popup Animation** — Smooth UI popup animations

### Citizen Equipment

- **Citizen Equipment System** — Assign equipment and tools to citizens
- **Equipment GUI** — Dedicated equipment management interface
- **Equipment Commands** — Server-side equipment management commands

### Container Role

- **Container Role Query** — Query container role information
- **HUD Display** — Role information HUD overlay

### Tourism & Immigration

- **Village Tourism** — Tourism system support
- **Town Immigration** — Immigration management service
- **Immigration GUI** — Immigration operation interface

### Building

- **Building Pack Deployment** — More flexible building pack deployment
- **Building Task Management** — Building task tracking and control

### Winery

- **Multi-Block Winery** — Industrial control box brewing various beverages
- **Multiple Beverages** — Wine, ice wine, honey wine, plum wine, sweet berry wine, rum, sakura wine, etc.
- **Kaleidoscope Tavern Compatible** — Uses KT drink items as products

### Compatibility

- **Xaero's World Map** — Ore vein highlighting and world map markers
- **LetFishLove Reborn** — Fish breeding and roe hatching
- **Tide** — Tide fish ecology compatibility
- **Aquaculture** — Aquaculture compatibility
- **LDLib2** — Modern GUI framework
- **Vinery** — Grape crop compatibility (red/white/savanna/taiga)
- **Kaleidoscope Tavern** — Drink items and grapevine crop compatibility
- **Multi-Farm Mods** — Farmers Delight, Farm & Charm, Cultural Delights, etc.

# XY2407 NSUK Addition

**NeoForge 1.21.1** — 为 Sim-U-Kraft 添加养殖、采矿、矿脉、城市升级、自动补货、酿酒厂等丰富功能的拓展模组。

| 属性 | 值 |
|------|-----|
| **模组 ID** | `xy2407_nsuk_addition` |
| **平台** | NeoForge 1.21.1 |
| **作者** | xingying2407 |
| **许可** | GPL-3.0 |

***

## 功能特性

### 养殖系统

- **养殖控制箱** — 管理动物与鱼类繁殖的多方块控制器
- **鱼类繁殖支持** — 兼容 LetFishLove Reborn、Tide、Aquaculture 的鱼类繁殖
- **鱼卵方块** — 动态鱼卵方块，支持多种鱼类
- **繁殖定义加载器** — 通过 JSON 配置文件自定义繁殖配方
- **建筑部署** — 自动部署鱼塘与养殖场建筑包

### 采矿系统

- **采矿控制箱** — 多方块采矿控制器
- **采矿菜单** — 完整 GUI 菜单（含 LDLib2 UI）
- **玩家手动采矿** — 手动采矿功能支持

### 矿脉系统

- **矿脉分布** — 世界生成时自然分布矿脉
- **矿脉发现** — 玩家可发现并登记矿脉
- **矿脉高亮** — 兼容 Xaero's World Map 的矿脉高亮显示
- **富铁矿石渲染** — 富铁矿 overlays 渲染
- **掉落处理** — 矿脉特有的掉落物处理

### 城市升级

- **城市等级系统** — 支持多级城市升级
- **升级需求** — 人口、资源、资金等条件检查
- **升级面板** — 替换原版 Sim-U-Kraft 升级界面
- **核心迁移** — 支持移动城市核心位置

### 自动补货

- **工业控制箱自动补货** — 自动补充工业控制箱的原料与燃料
- **商业控制箱自动补货** — 自动补货商业控制箱的商品
- **SQLite 持久化** — 补货配置持久化存储

### 农田与作物扩展

- **高杆作物支持** — 支持甘蔗、竹子等高杆作物
- **藤蔓与灌木** — 葡萄藤、葡萄灌木等
- **水稻** — 水稻种植支持
- **右键收获** — 一键右键收割兼容作物
- **多方块作物支持** — 兼容 Farmers Delight、Farm & Charm、Cultural Delights、Kaleidoscope Cookery 等模组作物

### 侧边栏 HUD

- **信息侧边栏** — 可配置的屏幕侧边信息显示
- **城市数据面板** — 显示城市人口、资金等统计数据
- **建筑统计** — 各类建筑的统计数据
- **弹出动画** — 平滑的 UI 弹出动画

### 市民装备

- **市民装备系统** — 为市民分配装备与工具
- **装备 GUI** — 专用的装备管理界面
- **装备命令** — 服务端装备管理命令

### 容器角色

- **容器角色查询** — 查询容器角色信息
- **HUD 显示** — 角色信息 HUD 叠加层

### 旅游与移民

- **村庄旅游** — 旅游系统支持
- **小镇移民** — 移民管理服务
- **移民 GUI** — 移民操作界面

### 建筑系统

- **建筑包部署服务** — 更灵活的建筑包部署机制
- **建筑任务管理** — 建筑任务跟踪与控制

### 酿酒厂

- **多方块酿酒厂** — 工业控制箱酿造多种酒类
- **多品种酒类** — 葡萄酒、冰酒、蜂蜜酒、梅酒、甜浆果酒、朗姆酒、樱花葡萄酒等
- **Kaleidoscope Tavern 兼容** — 使用 KT 酒类物品作为产物

### 兼容性整合

- **Xaero's World Map** — 矿脉高亮与地图标记
- **LetFishLove Reborn** — 鱼类繁殖与鱼卵孵化
- **Tide** — 鱼类生态兼容
- **Aquaculture** — 水产养殖兼容
- **LDLib2** — 现代化 GUI 框架
- **Vinery** — 葡萄作物兼容（红/白/热带/针叶林）
- **Kaleidoscope Tavern** — 酒类物品与葡萄藤作物兼容
- **多种农业模组** — Farmers Delight、Farm & Charm、Cultural Delights 等
